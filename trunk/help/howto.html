How to Configure the Contact Form 7 Newsletter Plugin

Learn how to connect a Contact Form 7 form to Constant Contact. Make sure you already have installed and activated the Contact Form 7 plugin.

Search for the plugin in the WordPress Plugins page.

Install the plugin

Click Install Now to install the plugin.

Activate the plugin

Click "Activate Plugin"

Install and activate the plugin

Go to the Contact Form 7 > Constant Contact settings page

Click on Contact in the WordPress admin sidebar. Then click on the Constant Contact link.

The settings page will look like this

Save your Constant Contact username and password

Once you have entered this information, save the form.

If your settings are correct, you will see this message

If your settings are NOT correct, you will see this message

Choose a Contact Form 7 form to edit

Click Generate Tag and select Constant Contact Lists

Configure the form by following on-screen instructions

This is just a sample configuration. Configure your form as you want!

Copy the code shown

Paste the code into your form where you want the signup to appear.

Check the "Send form entries to Constant Contact" checkbox

Map integration fields to the form fields

For each Contact Form 7 form field that you want sent to Constant Contact, find the Integration Field related to it. The email address field is required.

Save the form

Once the form is saved, you're all set!

You can view integration status on the Contact Form 7 Forms page.

When the form is connected to Constant Contact, you will see the Constant Contact icon next to the form name.
